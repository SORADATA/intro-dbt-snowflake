dbt adapté aux données de pipeline de type ELT ( Extract Load Transform) cad qu'on télécharge directement dans la datawarehouse puis on effectue la transformation. 

dbt core est la version utilisé en local..
dbt est un outil modern stack qui permet la trasnformation des données avec SQL beaucoup plus pratique pour les Data Analysts

- Un DataLake : Un entrepot de données ou on peut mettre tout type de données notamment vidéo, MP3 , CSV etc..
- Un datawarehouse : Un entreprot qui stocker spécifiquement des données typées tabulaire(variables, individus) notamment du parquet 





code pour lancer dbt core il suffit de faire :
- pip install debt et pip install dbt-snowflake
- dbt init
